第2章 距離空間 — 開球

▸ 索引へ戻る

ℝ² 上で、選んだ距離 d による 開球 Br(c) = { x ∈ ℝ² : d(c, x) < r } の形を可視化します。中心 c、半径 r、距離の種類を自由に変えられます。

距離の選択

⚠ p < 1 では三角不等式が成り立たず距離ではありません (図示のみ・球は凹む)。

パラメータ

表示

L¹   L²   L∞   Lᵖ

定義と要点

距離の比較(本講では 1 ≤ p ≤ ∞):

  • L¹: d1(x, y) = |x₁−y₁| + |x₂−y₂| → 開球は菱形
  • L²: d2(x, y) = √((x₁−y₁)² + (x₂−y₂)²) → 円
  • L∞: d∞(x, y) = max(|x₁−y₁|, |x₂−y₂|) → 正方形
  • Lp: dp(x, y) = (Σ |xi−yi|p)1/p

大事なポイント: ℝⁿ 上の Lp 距離 (1 ≤ p ≤ ∞) はすべて同じ位相を定めます。形は違っても、どの Lp 球も他の Lq 球と互いに入れ子にできるため、開集合の集まり(=位相)は同じです。「比較」をオンにすると同じ r で形だけ変わる様子が見えます。
